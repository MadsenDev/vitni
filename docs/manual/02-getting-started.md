# Getting Started

## Creating a Project

Open Vitni and click **Start New Investigation** on the welcome screen. Give the project a name. Vitni creates a project package on disk — a folder containing the case database, any attached media, and exported reports.

You'll be asked to choose an **investigation profile** which sets default node type ordering:

| Profile | Best for | Default order |
|---|---|---|
| General Investigations | General case work | People → Orgs → Tech → Evidence |
| Cyber / OSINT | Digital investigations | Tech → People → Orgs → Evidence |
| Casework / LE | Law enforcement case files | People → Evidence → Orgs → Tech |

You can change the profile at any time in Settings.

## Opening an Existing Project

Click **Open Existing Case** on the welcome screen, or use **File → Open** from the menu bar. Recent projects are listed on the welcome screen for quick access.
