# Node Types

Vitni organises nodes into four categories.

## People

**Person** — A human subject. Key properties: first name, last name, aliases, birth date, nationality, email, phone, occupation, national ID, passport, investigative role (Subject, Witness, Victim, Associate, Investigator, etc.), photo.

## Organizations

**Organization** — A company, government body, NGO, or other group entity. Key properties: legal name, trade name (DBA), organization type, sector, industry, jurisdiction, registration number, tax ID, website, founding date, status.

## Technology

| Type | What it represents | Key property |
|---|---|---|
| Domain | A registered internet domain | Domain name, registrar, registrant |
| Website | A specific URL or web property | URL, hosting provider |
| Online Account | A social media or web service account | Platform, handle/username |
| Email Address | An email address | Address, provider |
| Phone Number | A phone number | Number, carrier, type |
| Device | A physical or virtual computing device | Device type, serial, IMEI |
| IP Address | An IPv4 or IPv6 address | Address, type (static/dynamic) |
| Infrastructure | A server, network segment, or hosting entity | Provider, ASN |
| Crypto Wallet | A blockchain wallet address | Address, blockchain, currency |

## Evidence

| Type | What it represents |
|---|---|
| Evidence Item | Generic physical or digital evidence |
| Document | A written or printed record |
| Identity Document | Passport, ID card, driver's licence |
| Communication | A message, call log, or correspondence |
| Media | A photo, video, or audio recording |
| Financial Account | A bank account, card, or payment account |
| Financial Transaction | A single money movement |
| Event | A scheduled or identified occurrence |
| Incident | A specific harmful or notable event |
| Location | A physical place or address |
| Vehicle | A car, truck, or other road vehicle |
| Aircraft | A plane, helicopter, or drone |
| Case | A formal case or investigation record |
