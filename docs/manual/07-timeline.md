# Timeline Workspace

Switch to Timeline from the workspace switcher at the top of the screen.

The Timeline shows nodes with date-typed properties arranged in chronological order. Events, incidents, communications, financial transactions, and any entity with a relevant date field appear on the timeline.

Use the timeline to:
- Verify the chronological coherence of events
- Spot gaps in the timeline where activity is unexplained
- Identify overlaps between actors at key moments

The timeline is read-only. You cannot edit node data from this view — return to Investigation to do that.
