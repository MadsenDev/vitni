# Command Palette and Shortcuts

## Command Palette (`⌘K` / `Ctrl+K`)

Opens a searchable palette with every action available in the app. Type anything to filter. Prefix with `>` to search commands only.

Default quick-access groups when empty:

- **Navigate** — Switch to Investigation, Timeline, or Review
- **Create** — Add Person, Organization, Domain, Evidence, Assertion, Source
- **Graph** — Fit to screen, Relationship tool, Toggle filters
- **File** — Export report, Settings

## Search (`⌘F` / `Ctrl+F`)

Full-text search across all nodes, relationships, assertions, and sources in the project. Results show title, type, and context. Selecting a result navigates to that entity on the graph.

## Keyboard Shortcuts

| Shortcut | Action |
|---|---|
| `⌘K` / `Ctrl+K` | Open Command Palette |
| `⌘F` / `Ctrl+F` | Open search |
| `Ctrl+Shift+F` | Fit graph to screen |
| `Ctrl+Shift+Z` | Zoom to selection |
| `Ctrl+Shift+C` | Center selection |
| `Esc` | Close any open palette or modal |
