# Settings and Personalization

Open Settings via **⌘K → Settings** or the menu bar.

## General

- **Default workspace** — Which workspace opens when you load a project (Investigation, Timeline, or Review)
- **Investigation profile** — Reorders node types in the palette based on your use case
- **Restore saved view on open** — Whether to restore the last used saved view when reopening a project
- **Default sidebar tab** — Whether the Inspector opens to Nodes or AI by default
- **Auto-hide inspector when idle** — Collapses the right sidebar when nothing is selected

## Graph

- **Show node labels** — Toggle text labels on graph nodes
- **Show node images** — Show profile photos on Person nodes
- **Auto-layout preset** — Automatically re-run a layout preset when new nodes are added

## Assertions

- **Default relationship confidence** — The confidence level pre-filled when creating a new relationship
- **Assertion field automation** — Controls whether entering a field value automatically creates a linked assertion (*Auto*, *Prompt*, or *Manual*)

## Appearance

- **Theme** — Visual colour theme for the interface
- **Icon pack** — The icon set used for node type icons
- **UI density** — Comfortable or compact layout
- **Motion preference** — Reduced, standard, or enhanced animations
