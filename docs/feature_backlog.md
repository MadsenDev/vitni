# Vitni Feature Backlog (remaining items)

Only includes items not already implemented in the app.

## 1) Core Interaction
- Zoom to Selection + Fit to Screen commands
- Undo/Redo: history of edits for safe exploration
- Lasso/Multi-select improvements: shift-add, invert, bounding box polish

## 2) Investigation / Analysis
- Find/Highlight: search nodes/edges, auto-zoom/flash results (Ctrl+F)
- Path Finder: shortest/all paths between two nodes, visual highlight
- Filter/Focus: by type, confidence, source; quick visibility toggles
- Group/Cluster: temporary groupings; collapse/expand cluster nodes
- Snapshot/Bookmark: save camera + visible nodes; quick recall
- Timeline: filter/animate visible nodes by date range
- Compare: diff two project versions (added/removed nodes/edges)
- Annotation notes: freeform sticky notes/highlights on canvas
- Verification panel: batch-toggle assertion confidence

## 3) AI Assistance
- Entity suggestions from text/docs (non-destructive proposals)
- Relation inference suggestions (co-occurrence/metadata based)
- Summarizer for selected subgraph → note
- Cluster naming (lightweight NLP/embeddings)
- Prompt console: natural-language commands to filter/focus

## 4) Layout / Visualization
- Center Selection command
- Color map: color by type, confidence, source count + legend
- Minimap overview
- Screenshot/Export: PNG (and copy-to-clipboard)

## 5) Case Management & Integrity
- File integrity verification: re-hash + mismatch warning
- Version snapshots: manual checkpoints for later diff
- Export/Report: local PDF/JSON case summary

## 6) Nice-to-Have / Later
- Script macros (à la Maltego Machines)
- Collaboration (P2P/E2EE shared view)
- Transform sandbox (manage/install/run transforms)
- Quick-Note tool (plain notes on canvas)
- Pin/Lock node positions
